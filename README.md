Marina Fleet – Yacht & Boat Booking Web Interface
Marina Fleet is a fully responsive front-end website designed to showcase luxury yachts and boat rental services. The website provides users with an elegant and user-friendly interface to explore various yachts, view pricing plans, check discounts, and get in touch with the company. It is built using HTML, CSS, Bootstrap, and JavaScript to ensure a smooth and visually appealing experience across all devices.

⚠️ Note: This is a simple static website with no backend functionality. All data and interactions are front-end only — no server side processing or database is involved.

Features:
Home Page – Engaging hero section with a luxury theme.
Gallery – Visual display of yachts and experiences.
Sailing & Luxury Listings – Categorized yacht collections with pricing and discounts.
Pricing Plans – Multiple rental packages with features and select options.
Contact Section – Inquiry form with phone, email, and Google Map integration.
Team Section – Meet the team behind the brand.
Fully Responsive – Built with Bootstrap for mobile, tablet, and desktop compatibility.
Modern UI – Clean layout with hover effects and interactive elements.

Technologies Used:
HTML5 – Structure and content
CSS3 – Styling and animations
Bootstrap 5 – Responsive grid and components
JavaScript – Interactivity (e.g., discount filters, form validation, etc.)

Purpose:
This project was developed to strengthen front-end development skills and demonstrate the ability to build a complete, responsive website from scratch purely static, with no backend involvement.
